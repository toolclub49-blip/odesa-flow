# Odesa Flow CRM

Next.js + Firebase Firestore версия CRM для заказов, базы клиентов и печати бирок.

## Что уже есть

- Добавление заказов через парсер сырого текста
- База клиентов
- Редактирование заказов и клиентов
- RU / UA интерфейс
- Печать бирки
- JSON импорт / экспорт
- Firebase Firestore синхронизация
- Локальный fallback, если Firebase ещё не настроен

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Скопировать `.env.example` в `.env.local`

3. Вставить Firebase Web config из проекта Firebase

4. Запустить dev-сервер:

```bash
npm run dev
```

## Firebase

Нужно создать Firestore Database и включить тестовые правила или свои безопасные правила.

Минимально:

- коллекция `orders`
- коллекция `clients`
